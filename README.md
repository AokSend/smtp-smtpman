# SMTPman - 专业SMTP邮件发送服务平台

SMTPman 是一款基于自研技术构建的高性能邮件推送服务平台，专注于为客户提供稳定、高效、低成本的 SMTP 邮件发送服务。我们致力于帮助开发者、企业和个人用户轻松集成邮件功能，无论是事务性通知、营销推广还是外贸拓客，SMTPman 都能提供专业级的解决方案。

## 🎯 服务场景

### A. 触发式邮件（推荐）
- **验证码邮件** - 用户注册、登录验证
- **安全通知** - 密码重置、账户异常提醒
- **业务提醒** - 余额变动、VIP到期通知、订单确认
- **系统通知** - 账号认证、服务状态更新

### B. 营销式邮件（推荐）
- **会员营销** - 新品发布、促销活动
- **新闻订阅** - 定期资讯、产品更新
- **客户维护** - 会员关怀、满意度调查

### C. 陌生式邮件（高级功能）
- **外贸拓客** - 海外客户开发
- **市场推广** - 产品推介、品牌宣传
- **活动邀请** - 行业会议、产品发布会

## 📋 快速开始

### 新手四步上手

1. **添加域名**
   - 准备您自己的域名（建议使用独立子域名）
   - 在 SMTPman 平台完成域名验证

2. **配置解析**
   - 按照指引添加 DNS 记录
   - 支持 SPF、DKIM、DMARC 等认证配置

3. **创建账户**
   - 生成专属 SMTP 账户信息
   - 测试 SMTP 连接性

4. **集成使用**
   - 将 SMTP 配置集成到您的系统
   - 开始发送邮件
